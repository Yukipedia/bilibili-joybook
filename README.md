## [使用说明Wiki](https://github.com/Yukipedia/bilibili-joybook/wiki)

## 本地构建
- 安装 [Node.js](https://nodejs.org) Current或者LTS版本
- cd bilibili_joybook
- 安装依赖 npm install 或者 yarn
- 构建生产版本使用 yarn build 或者 npm run build
- 构建开发版本使用 yarn serve 或者 npm run serve
